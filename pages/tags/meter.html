Tag <meter>

Anterior | Próximo

Definições e uso

A tag <meter> define uma medida escalar dentro de um intervalo conhecido ou um valor fracionário. Isso também é conhecido como um medidor.

Exemplos: Uso de disco, relevância de um resultado de consulta, etc.

Observação: A tag <meter> não deve ser usada para indicar o progresso (como em uma barra de progresso). Para barras de progresso, use a tag <progress>

Dica: Sempre adicione a tag <label> para melhores práticas de acessibilidade.

Exemplos 


<label for="disk_c">Uso de disco C:</label>
<meter id="disk_c" value="2" min="0" max="10">2 de 10</meter><br>

<label for="disk_d">Uso de disco D:</label>
<meter id="disk_d" value="0.6">60%</meter>

Atributos

Atributo Valor Descrição
form form_id Especifica a qual formulário o elemento <meter> pertence
high number Especifica o intervalo que é considerado como um valor alto
low number Especifica o intervalo que é considerado como um valor baixo
max number Especifica o valor máximo do intervalo
min number Especifica o valor mínimo do intervalo. O valor padrão é 0.
optimun number Especifica qual valor é o valor ideal para o medidor
value number Obrigatório. Especifica o valor atual do medidor
