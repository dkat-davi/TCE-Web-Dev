Tag <header>

Anterior | Próximo

Definições e uso

O elemento <header> representa um container para conteúdo introdutório ou um set de links navegacionais

Um elemento <header> tipicamente contém

  • Um ou mais elementos de cabeçalho (<h1>-<h6>)
  • Logo ou ícone
  • Informações autorais

Nota: Você pode ter diversos <header> elementos em um documento HTML. Porém, <header> não pode ser aplicado dentro de um <footer>, <address> ou outro elemento <header>

Exemplos 


<article>
    <header>
        <h1>A heading here</h1>
        <p>Posted by John Doe</p>
        <p>Some additional information here</p>
        <hr>
    </header>
    <p>Lorem Ipsum dolor set amet....</p>
</article>
