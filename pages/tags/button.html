Definições e uso

A tag <button> define um botão clicável

Dentro de um elemento <button> você pode inserir texto (e tags como <i>, <b>, <strong>, <img>, etc.). Isso não é possível com um botão criado com o elemento <input>

Sempre defina o atributo type para o elemento <button>,para mostrar ao navegador qual o tipo deste botão.

Exemplos 


<button type="button">Click Me!</button>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag button</h1>

<button type="button">Click Me!</button>

</body>
</html>
