Tag <table>

Anterior | Próximo

Definições e uso

A tag <table> define uma tabela HTML.

Uma tabela HTML consiste em um elemento <table> e um ou mais elementos <tr>, <th> e <td>

O elemento <tr> define uma linha da tabela, o elemento <th> define um cabeçalho de tabela e o elemento <td> define uma célula da tabela

Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>

Exemplos 


<table>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
</table>

Exemplo


<!DOCTYPE html>
<html>
<head>
<style>
    table, th, td {
        border: 1px solid black;
    }
</style>
</head>
<body>
                                
<h1>Tag table</h1>
                                
<table>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
    <tr>
        <td>February</td>
        <td>$80</td>
    </tr>
</table>
                                
</body>
</html>
