Tag <iframe>

Anterior | Próximo

Definições e uso

A tag <iframe> especifica um frame em linha.

Um frame em linha é usado para embutir outro documento ao documento HTML atual.

Dica: Use CSS para estilizar a tag <iframe>.

Dica: É uma boa prática incluir um atributo de título para a tag <iframe>. Isso é usado por leitores de tela para ler o que o conteúdo do <iframe> é.

Exemplos 


<iframe src="https://www.ifnmg.edu.br" title="Site do Ifnmg">
</iframe>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag iframe</h1>

<iframe src="https://www.ifnmg.edu.br" title="Site do Ifnmg">
</iframe>

</body>
</html>

Atributos

Atributo Valor Descrição
allow nenhum Especifica uma política de recursos para o elemento <iframe>
allowfullscreen true/false Definir como verdadeiro se o elemento <iframe> puder ativar o modo de tela cheia ao chamar o método requestFullscreen().
allowpaymentrequest true/false Definir como verdadeiro se um <iframe> de origem cruzada deve ser permitido a invocar a API de Pedido de Pagamento (Payment Request API)
height pixels Especifica a altura de um <iframe>. Altura padrão é de 150 pixels
loading eager/lazy Especifica se um navegador deve carregar um <iframe> imediatamente ou adiar o carregamento dos iframes até que algumas condições sejam atendidas.
name text Especifica o nome de um <iframe>
referrerpolicy no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, same-origin, strict-origin-when-cross-origin unsafe-url Especifica qual informação de referência (referrer) deve ser enviada ao buscar o <iframe>
sandbox allow-forms, allow-pointer-lock, allow-popups, allow-same-origin, allow-scripts, allow-top-navigation Ativa um conjunto adicional de restrições para o conteúdo em um <iframe>.
URL src Especifica o endereço do documento a ser embutido no elemento <iframe>
srcdoc HTML_code Especifica o conteúdo HTML da página a ser exibido no elemento <iframe>
Width Pixels Especifica a largura de um elemento <iframe>. Valor padrão é 300 pixels.
