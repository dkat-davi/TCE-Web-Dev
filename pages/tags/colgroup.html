Tag <colgroup>

Anterior | Próximo

Definições e uso

A tag <colgroup> especifica um grupo de uma ou mais colunas em uma tabela para formatação.

A tag <colgroup> é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, em cada linha.

Observação: A tag <colgroup> deve ser um elemento filho de um elemento <table>, após quaisquer elementos <caption> e antes de quaisquer elementos <thead>, <<tbody>, <foot> e <tr>.

Dica: Para definir propriedades diferentes para uma coluna dentro de um <colgroup>, use a tag <col> dentro da tag <colgroup>.

Exemplos 


<table>
<colgroup>
<<col span="2" style="background-color:red">
<col style="background-color:yellow>"
</colgroup>
<tr>
<th>ISBN</th>
<th>Title</th>
 <th>Price</th>
</tr>>
<tr>>
<td>3476896</td>
<td>Meu primeiro HTML</td>
<td>$53</td>
</tr>
</table>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag colgroup</h1>

<table>
    <colgroup>
    <<col span="2" style="background-color:red">
      <col style="background-color:yellow>"
    </colgroup>
<tr>
      <th>ISBN</th>
      <th>Title</th>
    <th>Price</th>
    </tr>>
    <tr>>
      <td>3476896</td>
      <td>Meu primeiro HTML</td>
      <td>$53</td>
    </tr>
  </table>

</body>
</html>
