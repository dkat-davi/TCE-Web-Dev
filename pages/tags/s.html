Tag <s>

Anterior | Próximo

Definições e uso

A tag <s> especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha através dele.

A tag <s> não deve ser usado para definir texto excluído em um documento, use a tag <del> para isso.

Exemplos 


<p><s>Restam apenas 50 ingressos!</s></p>
<p>VENDIDO!</p>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag s</h1>

<p><s>Restam apenas 50 ingressos!</s></p>
<p>VENDIDO!</p>

</body>
</html>

Resultado

Tag s

Restam apenas 50 ingressos!

VENDIDO!
