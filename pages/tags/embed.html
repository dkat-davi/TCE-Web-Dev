Tag <embed>

Anterior | Próximo

Definições e uso

A tag <embed> define um contêiner para um recurso externo, como uma página web, uma imagem, um reprodutor de mídia ou um aplicativo de plug-in.

Exemplos 


<embed type="image/jpg" src="Royal_Palace,_Rabat.jpg" width="300" height="200">

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag embed</h1>

<embed type="image/jpg" Royal_Palace,_Rabat.jpg" width="300" height="200">

</body>
</html>

Tag embed

Atributos

Atributo Valor Descrição
Height Pixels Especifica a altura do conteúdo embutido
Src URL Especifica o endereço do arquivo externo para embutido
Type Type Especifica o tipo de mídia do conteúdo embutido
Width Pixels Especifica a largura do conteúdo embutido
