Tags <h1> ao <h6>

Anterior | Próximo

Definições e uso

As tags <h1> a <h6> são usadas para definir os cabeçalhos HTML.

A tag <h1> define o cabeçalho mais importante, enquantolt <h6> define o cabeçalho menos importante.

Nota: use apenas um <h1> por página, pois este deve representar o cabeçalho/assunto principal para toda a página.

Além disso, não pule níveis de cabeçalho - comece com <h1>, depois use <h2>, e assim por diante.

Exemplos 


<h1>Esse é o cabeçalho 1</h1>
<h2>Esse é o cabeçalho 2</h2>
<h3>Esse é o cabeçalho 3</h3>
<h4>Esse é o cabeçalho 4</h4>
<h5>Esse é o cabeçalho 5</h5>
<h6>Esse é o cabeçalho 6</h6>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Esse é o cabeçalho 1</h1>
<h2>Esse é o cabeçalho 2</h2>
<h3>Esse é o cabeçalho 3</h3>
<h4>Esse é o cabeçalho 4</h4>
<h5>Esse é o cabeçalho 5</h5>
<h6>Esse é o cabeçalho 6</h6>

</body>
</html>

Resultado

Esse é o cabeçalho 1

Esse é o cabeçalho 2

Esse é o cabeçalho 3

Esse é o cabeçalho 4

Esse é o cabeçalho 5
Esse é o cabeçalho 6
