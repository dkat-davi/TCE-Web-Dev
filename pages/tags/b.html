Tag <b>

Anterior | Próximo

Definições e uso

The <b> tag specifies bold text without any extra importance.

Exemplos 


<p>This is normal text - <b>and this is bold text</b>.</p>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag b</h1>

<p>This is normal text - <b>and this is bold text</b>.</p>

</body>
</html>

Resultado

Tag b

This is normal text - and this is bold text.
