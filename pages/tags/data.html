Tag <data>

Anterior | Próximo

Definições e uso

A tag <data> é usada para adicionar uma tradução legível por máquina de um conteúdo fornecido.

Este elemento fornece um valor legível por máquina para processadores de dados e um valor legível para humanos para renderização em um navegador.

Dica: Se o conteúdo estiver relacionado a data ou hora, use o elemento <time> em seu lugar.

Exemplos 


<ul>
    <li><data value="21053">Cherry Tomato</data></li>
    <li><data value="21054">Beef Tomato</data></li>
    <li><data value="21055">Snack Tomato</data></li>
</ul>
