Tag <link>

Anterior | Próximo

Definições e uso

A tag <link> define uma relação entre o documento atual e um recurso externo.

A tag <link> é usada mais frequentemente para vincular à folhas de estilo externas ou para adicionar um favicon ao seu site.

O elemento <link> é um elemento vazio, contendo apenas atributos.

Exemplos 


<head>
<link rel="stylesheet" href="styles.css">
</head>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag link</h1>

<head>
<link rel="stylesheet" href="styles.css">
</head>

</body>
</html>

Atributos

Atributo Valor Descrição
crossorigin anonymous, use-credentials Especifica como o elemento lida com solicitações Cross-Origin
href URL Especifica a localização do documento vinculado
hreflang language_code Especifica o idioma do texto no documento vinculado
media media_query Especifica em qual dispositivo o documento vinculado será exibido
referrerpolicy no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, unsafe-url Especifica qual referenciador usar ao buscar o recurso
rel alternate, author, dns-prefetch, help, icon, license, next, pingback, preconnect, prefetch, preload, prerender, prev, search, stylesheet Obrigatório. Especifica a relação entre o documento atual e o documento vinculado
sizes HeightxWidth, any Especifica o tamanho do recurso vinculado. Somente para rel="icon"
title Define uma folha de estilo preferida ou alternativa
type/td> media_type Especifica o tipo de mídia do documento vinculado
