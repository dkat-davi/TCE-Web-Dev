Tag <samp>

Anterior | Próximo

Definições e uso

A tag <samp> é usada para definir uma amostra de output de um programa computacional. O conteúdo dentro é mostrado na fonte monospace padrão do navegador

Dica: Embora essa tag não seja deprecada, é possível atingir um efeito mais rico usando CSS

Exemplos 


<p>Message from my computer:</p>


<p><samp>File not found.
Press F1 to continue</samp></p>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag samp</h1>

<p>Message from my computer:</p>


<p><samp>File not found.
Press F1 to continue</samp></p>

</body>
</html>

Resultado

Tag samp

Message from my computer:

File not found. Press F1 to continue
