Tag <summary>

Anterior | Próximo

Definições e uso

A tag <summary> define um cabeçalho visível para o elemento <details>. O cabeçalho pode ser clicado para exibir/ocultar os detalhes

Nota: o elemento <summary> deve ser o primeiro elemento filho do elemento <details>.

Exemplos 


<details>
    <summary>Epcot Center</summary>
    <p>Epcot é um parque temático no Walt Disney World Resort com atrações emocionantes, pavilhões internacionais, fogos de artifício premiados e eventos especiais sazonais.</p>
</details>

Exemplo


<!DOCTYPE html>
<html>
<body>

</details>

</body>
</html>
