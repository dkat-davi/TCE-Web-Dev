Tag <blockquote>

Anterior | Próximo

Definições e uso

A tag <blockquote> especifica uma seção que é citada de outra fonte.

Os navegadores geralmente irão indentar os elementos <blockquote> (é necessário utilizar CSS para remoção da indentação)

Dica: use <q> para breve citações em linha

Exemplos 


<blockquote cite="http://www.worldwildlife.org/who/index.html">
W3Schools é um site educacional voltado ao aprendizado de tecnologias web. Seu conteúdo inclui tutoriais e referências 
relacionadas a HTML, CSS, JavaScript, JSON, PHP, Python, AngularJS, SQL, Bootstrap, Node.js, jQuery, XQuery, AJAX, XML, e Java.
</blockquote>
