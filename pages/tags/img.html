Tag <img>

Definições e uso

A tag <img> é usada para incorporar uma imagem em uma página HTML.

As imagens não são tecnicamente inseridas em uma página da web; as imagens são vinculadas às páginas da web. A tag <img> cria um espaço reservado para a imagem referenciada.

A tag <img> tem dois atributos obrigatórios:

src - Especifica o caminho para a imagem
alt - Especifica um texto alternativo para a imagem, caso a imagem, por algum motivo, não possa ser exibida.

Nota: sempre especifique a largura e a altura de uma imagem. Se a largura e a altura não forem especificadas, a página poderá piscar enquanto a imagem é carregada.

Dica: para vincular uma imagem a outro documento, basta aninhar a tag <img> dentro da tag <a> (veja o exemplo abaixo).

Exemplos 


<img src="img_girl.jpg" alt="Garota com uma jaqueta" width="500" height="600">

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag img</h1>

<img src="img_girl.jpg" alt="Garota com uma jaqueta" width="500" height="600">

</body>
</html>
