Tag <dialog>

Anterior | Próximo

Definições e uso

A tag <dialog> define uma caixa de diálogo ou sub-janela

O elemento <dialog> facilita a criação de caixas de diálogo pop-up e modais em uma página da web

Exemplos 


<dialog open>Essa é uma janela de dialog aberta</dialog>

Exemplo


<!DOCTYPE html>
<html>
<body>
                                
<h1>Tag dialog</h1>
                                
<p>Texto aleatório</p>
                                
<p>Texto aleatório</p>
                                
<dialog open>Essa é uma janela de dialog aberta</dialog>
                                
<p>Texto aleatório</p>
                                
<p>Texto aleatório</p>
                                
</body>
</html>

Atributos

Atributo Valor Descrição
open open Especifica que o elemento de diálogo está ativo e que o usuário pode interagir com ele.
