Tag <area>

Anterior | Próximo

Definições e uso

A tag <area> define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis). define uma relação entre o documento atual e um recurso externo.

Os elementos <area> estão sempre aninhados dentro de uma tag <map>.

Observação: o atributo usemap em <img> está associado ao atributo name do elemento <map> e cria um relacionamento entre a imagem e o mapa.

Exemplos 


<img src="workplace.jpg" alt="Workplace" usemap="#workmap" width="400" height="379">

<map name="workmap">
<area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
<area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
<area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
</map>
