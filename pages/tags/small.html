Tag <small>

Definições e uso

A tag <small> define textos menores (como copyrights e outros comentários de rodapé)

Dica: Embora essa tag não seja deprecada, é possível atingir um efeito mais rico usando CSS

Exemplos 


<p>Este é um texto normal</p>
<p><small>Este é um texto menor.</small></p>

Exemplo


<!DOCTYPE html>
<html>
<body>

<h1>Tag small</h1>

<p>Este é um texto normal</p>
<p><small>Este é um texto menor.</small></p>

</body>
</html>
