Tag <ol>

Anterior | Próximo

Definições e uso

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

A tag <li> é usada para definir cada item da lista.

Dica: Use CSS para estilizar listas.

Dica: Para uma lista não ordenada, use a tag <ul>.

Exemplos 


<ol>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ol>
                              
<ol start="50">
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ol>
