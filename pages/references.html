Descrição

<a>

Define um Hyperlink

<area>

Define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis)

<b>

Define texto em negrito

<blockquote>

Define uma seção que é citada de outra fonte

<br>

Define uma única quebra de linha

<button>

Define um botão clicável

<canvas>

Usada para desenhar gráficos via scripting (normalmente JavaScript)

<code>

Define um pedaço de código de computador

<colgroup>

Especifica um grupo de uma ou mais colunas em uma tabela para formatação

<data>

Adiciona uma tradução 'legível por máquina' de um conteúdo fornecido

<datalist>

Especifica uma lista de opções predefinidas para controles de entrada

<dialog>

Define uma caixa ou janela de diálogo

<div>

Define uma seção em um documento

<embed>

Define um contêiner para um recurso externo

<header>

Define um cabeçário para um documento ou seção

<h1> ao <h6>

Define cabeçalhos HTML (Títulos e Subtítulos)

<i>

Define uma parte do texto em uma voz ou humor alternativo

<iframe>

Define um frame em linha

<img>

Define uma imagem

<input>

Define um controle de entrada

<link>

Define a relação entre um documento e um recurso externo (mais usado para vincular folhas de estilo)

<meta>

Define metadados sobre um documento HTML

<meter>

Define uma medição escalar dentro de um intervalo conhecido (um medidor)

<nav>

<ol>

<script>

<small>

Define um texto menor

<sub>

Define o texto subscrito

<summary>

<svg>

Define um contêiner para gráficos SVG

<table>

Define uma tabela

<template>

Define um contêiner para o conteúdo que deve ser ocultado quando a página é carregada

<time>

Define um time específico, ou datetime

<tr>

<ul>
